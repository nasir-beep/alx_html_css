# CSS Basics Project

This project demonstrates basic CSS styling applied to HTML pages with responsive design and modern layout techniques.

## Project Structure
- `index.html` - Homepage with basic content
- `tweets.html` - Page with embedded tweet examples
- `base.css` - Base styles and CSS reset
- `styles.css` - Custom styles for the project

## Features
- **CSS Flexbox Layout** - Modern responsive layout system
- **Responsive Design** - Adapts to all screen sizes (desktop, tablet, mobile)
- **Modern Styling** - Gradients, shadows, transitions, and hover effects
- **Custom Typography** - Gradient text effects and proper hierarchy
- **Interactive Elements** - Smooth animations and transitions
- **Professional Color Scheme** - Complementary gray and red palette
